# Weather Dashboard

A modern weather dashboard built with **React**, **Vite**, and **Tailwind CSS**.  
It fetches real‑time weather data from the **OpenWeather API** and displays it in a clean, responsive interface.


## Features
- **City search**: Look up weather conditions for any location.
- **Live weather data**: Temperature, humidity, wind speed, and pressure.
- **Responsive design**: Styled with Tailwind CSS for mobile and desktop.
- **Fast performance**: Powered by Vite for rapid development and optimized builds.
- **Dynamic visuals**: Weather descriptions and condition icons.


## Tech Stack
- **React** → Component‑based UI
- **Vite** → Lightning‑fast dev server and bundler
- **Tailwind CSS** → Utility‑first styling
- **OpenWeather API** → Reliable weather data source
